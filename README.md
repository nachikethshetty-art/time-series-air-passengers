# Time Series Analysis of Airline Passengers

This project performs time series analysis on monthly airline passenger data from 1949 to 1960.

## Objective
To analyze trends and seasonality in airline passenger data and forecast future demand using ARIMA.

## Techniques Used
- Time Series Decomposition
- Moving Averages
- ARIMA Forecasting

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Statsmodels

## Results
The analysis shows a strong upward trend and clear yearly seasonality.  
The ARIMA model provides accurate short-term forecasts.
